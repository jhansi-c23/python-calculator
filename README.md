# Python Calculator

A simple calculator application built using Python. This project performs basic arithmetic operations based on user input.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- User-friendly menu-driven program

## Technologies Used
- Python

## Concepts Used
- Functions
- Conditional Statements
- User Input
- Arithmetic Operations

## How to Run

1. Install Python on your system.
2. Save the file as `calculator.py`
3. Run the program using:

```bash
python calculator.py
```

## Sample Output

```text

2.subtract
```
